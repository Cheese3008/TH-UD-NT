# UI Modern Update

Phiên bản này giữ nguyên logic phát nhạc / playlist của project gốc và thiết kế lại phần giao diện:

- Theme tối hiện đại dùng Material 3.
- Màn hình Bài hát có hero card, thống kê số bài hát, số nghệ sĩ, tìm kiếm và chip sắp xếp.
- Danh sách bài hát chuyển sang card bo góc, dễ nhìn hơn.
- Mini player được làm lại thành floating card có progress bar.
- Màn hình Now Playing có bố cục album lớn, progress card và control button hiện đại.
- Màn hình Playlist có summary card, empty state, playlist card mới và màn hình chi tiết playlist gọn hơn.
- Màn hình Cài đặt chia thành các card rõ chức năng: volume, shuffle, repeat, ghi chú test.

Sau khi giải nén, chạy:

```bash
flutter pub get
flutter run
```
